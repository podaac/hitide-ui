System Recommendations
To best experience the HiTIDE client, we recommend using one of the following WebGL enabled browsers:
  1. Google Chrome 18+
  2. Mozilla Firefox 4.0+
  3. Apple Safari 8+

Older browser versions will work, but you will not have the ability to experience the 3D globe. To check if your browser supports WebGL, you can visit https://get.webgl.org.

Internet Explorer Support: Although Internet Explorer 10+ is supported, we suggest using Chrome, Firefox or Safari for a better experience.
