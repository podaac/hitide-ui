Cloud Migration Message

HiTIDE is moving to a new URL:

https://hitide.podaac.earthdatacloud.nasa.gov/

We recommend using this new URL immediately, as the current HiTIDE URL you are visiting is scheduled to be retired in early 2023.

This is part of PODAAC's migration to the cloud. All new HiTIDE datasets will be hosted in the cloud, and existing HiTIDE datasets are currently being migrated to the cloud.

These cloud hosted datasets will only be visible at the new HiTIDE URL, and are indicated with a cloud icon (as seen below).

Remember to update bookmarks and links!
